## MarkDownの書き方
---
#### 〇文字効果  
　　・太字：**Bold**  

　　・イタリック：_Italic_  

　　・取り消し線：~~Strike Line~~  
　

#### 〇コード
```java
int i = 0; //コード
```  
　

#### 〇ブロッククオート
>Blockquotes  

　
　
#### 〇テーブル  
|header1|header2|header3|
|:--:|:--:|:--:|
|a|b|c|  

　
　
#### 〇Notes
{{% notice note %}}
Notes類はShortCodeを利用して記述します。  
NoteのほかにInfo, Tip, Warningがあります。
{{% /notice %}}

{{% notice info %}}
information
{{% /notice %}}

{{% notice tip %}}
Tip
{{% /notice %}}

{{% notice warning %}}
Warning
{{% /notice %}}
